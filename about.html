VISUALICER

A play on creator ICE's name: a deliberately over-editable visualizer built as a weekend SPA project.

What it does

VISUALICER turns local audio, artwork and timed captions into a responsive radial player. Nearly every visible choice can be tuned, producing a surprising number of combinations from a deliberately simple interface.

Private by design

Audio, images and lyric files are processed in your browser and never uploaded. Media object URLs exist only for the active page. Theme choices, layout preferences, metadata, lyrics and a non-media position may be stored in this browser using localStorage.

Built for detail

The project is an exercise in attention to every frontend detail: theme variants, typography, radial geometry, responsive composition, motion preferences, keyboard interaction and local media lifecycle.

How ICE uses it

VISUALICER is designed for frequent use alongside the SSS project, while remaining a self-contained single-page application anyone can open locally.

Quick start

  1. Open Edit and choose audio.
  2. Add artwork, metadata and lyrics if wanted.
  3. Pick a theme, mode, font and stroke treatment.
  4. Play, seek and tune the interface. Local settings restore next time.
